# openrcageverification
Age verification application for OpenRC init system in linux written in C++
There is no escape from the law this is for linux systems
running OpenRC who think they are bypassing age verification laws
this is setup to check if you are 18+ but can be modified for the U.S.
with a 21 age check if you look at the source code
This application is intended to help the
federal bureau of investigation and other law
enforcement agencies find out your legal age
